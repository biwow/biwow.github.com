# 部署智能合约

使用postman向节点rpc接口提交信息，部署合约需要的gas需要设置的高一些

```
{
    "jsonrpc": "2.0",
    "method": "eth_sendTransaction",
    "params": [
        {
            "from": "0x6e60f5243e1a3f0be3f407b5afe9e5395ee8aaaa",
            "gas": "0x3d0900",
            "gasPrice": "0x9184e72a000",
            "value": "0x0",
            "data": "0x60606040526000806014610100*****afa1fd553445400000000000000000000000000000000000000000000000000000000"
        }
    ],
    "id": 1
}
```

使用部署合约hash查询合约地址
```
{
    "jsonrpc": "2.0",
    "method": "eth_getTransactionReceipt",
    "params": ["0xc3aa3de6ae591f9d2566451be28804489d737505a9c783f5bbbc14a2621c1ac8"],
    "id": 1
}
```

部署的合约地址为：0xafe45e2209e45539b0f09dad323a77f1ba9d9b96
